# H2 Database Engine

<http://www.h2database.com/html/main.html>

## 简介：Java SQL 数据库

* 高速，开源，JDBC API
* 嵌入模式，服务器模式；内存数据库
* 基于浏览器管理
* 轻量，2 MB Jar 包
